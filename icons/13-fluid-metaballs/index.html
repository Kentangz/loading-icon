13 · Fluid Metaballs

An organic liquid gooey loader. As the satellite spheres pass through the central core dot, they merge, stretch, and sever in a realistic fluid animation.

💡 Gooey Effect Notice: The gooey effect utilizes CSS contrast filters. For it to render correctly, you must set the --metaballs-bg property to exactly match the solid background color of the surface behind the loader.

Quick usage

<!-- 1. Import the stylesheet -->
<link rel="stylesheet" href="fluid-metaballs.css" />

<!-- 2. Structure — requires exactly 3 span children -->
<div class="fluid-metaballs" role="status" aria-label="Loading">
  <span></span><span></span><span></span>
</div>

<!-- 3. Customizing the background color to match your layout -->
<div class="fluid-metaballs" style="--metaballs-bg: #ffffff; --metaballs-color: #1D9E75;">
  <span></span><span></span><span></span>
</div>
