A premium color-spectrum wheel spinner. A multi-spectral conic gradient is masked into a smooth ring shape, creating a gorgeous chromatic aberration rotation with zero SVG assets.

Quick usage

<!-- 1. Import the stylesheet -->
<link rel="stylesheet" href="chromatic-spinner.css" />

<!-- 2. Single element — no children needed -->
<div class="chromatic-spinner" role="status" aria-label="Loading"></div>

<!-- 3. Size + Glow modifiers -->
<div class="chromatic-spinner chromatic-spinner--lg chromatic-spinner--glow"></div>

<!-- 4. Full custom thickness & speed override -->
<div class="chromatic-spinner" style="
  --chromatic-size:      56px;
  --chromatic-thickness: 8px;
  --chromatic-speed:     0.7s;
"></div>
